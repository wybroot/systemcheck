#### 2.1.2 核心代码实现

```python
class SchedulerService:
    @staticmethod
    def add_inspection_job(job_id, cron_expr, server_ids=None):
        # 解析Cron表达式
        parts = cron_expr.split()
        trigger = CronTrigger(
            minute=parts[0],
            hour=parts[1],
            day=parts[2],
            month=parts[3],
            day_of_week=parts[4]
        )
        
        # 添加任务
        scheduler.add_job(
            func=SchedulerService.run_scheduled_inspection,
            trigger=trigger,
            id=job_id,
            args=[server_ids],
            replace_existing=True
        )
    
    @staticmethod
    def run_scheduled_inspection(server_ids):
        # 执行巡检
        results = InspectorService.run_batch_inspection(server_ids)
        
        # 检查异常并发送告警
        alerts = [r for r in results.get('success', []) 
                  if r.get('status') != 'OK']
        if alerts:
            NotifyService.send_inspection_alert(alerts)
```

#### 2.1.3 Cron表达式说明

| 表达式 | 说明 |
|--------|------|
| `0 8 * * *` | 每天早上8点 |
| `0 */2 * * *` | 每2小时 |
| `0 9,18 * * *` | 每天9点和18点 |
| `0 0 * * 1` | 每周一凌晨 |
| `0 0 1 * *` | 每月1号凌晨 |

#### 2.2.2 钉钉机器人实现

```python
@staticmethod
def send_dingtalk(config, message):
    webhook = config.get('webhook')
    secret = config.get('secret')
    
    # 签名验证
    if secret:
        timestamp = str(round(time.time() * 1000))
        string_to_sign = f"{timestamp}\n{secret}"
        hmac_code = hmac.new(
            secret.encode('utf-8'),
            string_to_sign.encode('utf-8'),
            digestmod=hashlib.sha256
        ).digest()
        sign = urllib.parse.quote_plus(base64.b64encode(hmac_code))
        webhook = f"{webhook}&timestamp={timestamp}&sign={sign}"
    
    # 发送消息
    data = {
        "msgtype": "text",
        "text": {"content": message}
    }
    response = requests.post(webhook, json=data, timeout=10)
    return response.json().get('errcode') == 0
```

```python
@staticmethod
def send_email(config, message):
    # 构建邮件
    msg = MIMEMultipart()
    msg['From'] = config.get('from_addr')
    msg['To'] = config.get('to_addrs')
    msg['Subject'] = f"【服务器巡检告警】{datetime.now():%Y-%m-%d %H:%M}"
    msg.attach(MIMEText(message, 'plain', 'utf-8'))
    
    # 发送邮件
    with smtplib.SMTP(config.get('smtp_host'), config.get('smtp_port')) as server:
        if config.get('smtp_user'):
            server.starttls()
            server.login(config.get('smtp_user'), config.get('smtp_pass'))
        server.sendmail(msg['From'], msg['To'].split(','), msg.as_string())
```

## 三、遇到的问题与解决方案

### 3.1 定时任务持久化问题

**问题描述：**
服务重启后定时任务丢失。

**解决方案：**
1. 任务配置保存到数据库settings表
2. 应用启动时从数据库加载任务配置
3. 使用APScheduler的SQLAlchemyJobStore持久化

```python
# 保存任务配置
setting = Setting(key=f'job_{job_id}', value=json.dumps(job_config))
db.session.add(setting)

# 启动时加载
def load_scheduled_jobs():
    jobs_setting = Setting.query.filter_by(key='scheduled_jobs').first()
    for job in json.loads(jobs_setting.value):
        SchedulerService.add_inspection_job(job['id'], job['cron'])
```

### 3.2 钉钉签名验证问题

**问题描述：**
钉钉机器人开启了签名验证，发送消息返回"sign not match"。

**解决方案：**
严格按照钉钉文档实现签名：
```python
timestamp = str(round(time.time() * 1000))
string_to_sign = f"{timestamp}\n{secret}"
hmac_code = hmac.new(
    secret.encode('utf-8'),
    string_to_sign.encode('utf-8'),
    digestmod=hashlib.sha256
).digest()
sign = urllib.parse.quote_plus(base64.b64encode(hmac_code))
```

### 3.3 Grafana数据库连接问题

**问题描述：**
Grafana连接PostgreSQL报错"connection refused"。

**解决方案：**
1. 使用Docker网络内的服务名作为host
2. 配置正确的端口和认证信息
3. 确保数据库已初始化

### 3.4 邮件发送超时问题

**问题描述：**
邮件发送时间过长导致请求超时。

**解决方案：**
1. 设置合理的超时时间（10秒）
2. 使用异步发送（后续优化）
3. 捕获异常并记录日志
